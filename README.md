# 🏁 F1 Race Finish Predictor

A simple tool to estimate a driver's final race position based on their team and grid start.

### 🔗 Live App
**[f1-finish-predictor.streamlit.app](https://f1-finish-predictor.streamlit.app/)**

---

### 🕹️ How it Works
* **Driver & Team Selection:** Choose any driver and constructor from the current grid.
* **Grid Position:** Use the interactive slider to set the starting position.
* **Instant Results:** Generate a predicted finish based on 2019–2024 data patterns.

---

### 📊 Data & Logic
A beginner-friendly data science project using Python to explore the relationship between starting grid positions and final race results.
